# Модуль Cookie 

Этот модуль выводит всплывающее уведомление о согласии на использование файлов cookie (Cookie) в правом нижнем углу сайта. Он является глобальным и автоматически подключается на все страницы в системе веб-панели.

## Сколько хранится информация и как изменить срок

По умолчанию согласие пользователя хранится **365 дней** (1 год). По истечении этого срока или при ручной очистке кук в браузере, баннер отобразится снова.

Вы можете изменить время хранения информации в файле:
`app/modules/module_block_cookie/assets/js/always.js`

отредактируйте параметр `expiryDays` на строке 5:

```javascript
const cookieName = 'cookie_consent_accepted';
const expiryDays = 365; // Срок действия в днях (например: 30, 90, 365)
```
